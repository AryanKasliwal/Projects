Basic layout of all pages:

JavaScript:

 Obtrusive and un-obtrusive event handlers:

  1) Used onlcick attributr on buttons and for the seating plan table on the tickets page.
  2) Used .addEventListener for playing the videos one after the other.
  3) Used .addEventListener for checking if the selected cinema has been changed in the dropdown.

 Functions:

  1) All functions are defined in extermal JavaScript files in all pages.
  2) All the functions made in external files are called in the html files to dynamically print the website.

Style sheets:

 CSS3 elements:

   1) '@keyframes logo' and 'animated-logo' id selectors are used to make the animation for moving the logo above the name of the company.
    Line: 86-99
   File: "Common styles.css"

  2) 'box-shadow' attribute used to make shadows for blue strip boxes and white boxes.
   Line: 13 & 24
   File: "Common styles.css"

   3) 'background-image: linear-gradient()' attribute used to make the colour of the blue strip gradient from blue to white.
   Line: Line: 21
   File: "Common styles.css"

   4) '@media print' attribute used to hide the header, footer and menu part of the page while printing.
   Line: Line: 1
   File: "Print page.css"

   5) 'page-break-after:always' attribute used to print every ticket on a new page.
   Line: Line: 11
   File: "Print page.css"

   There are a total of 5 style sheets used for all the pages.

 Common style sheet for all pages:

   - One file named "Common style.css" contains the style elements that apply to all pages.
   - 'menu' id selector that makes the position of the menu sticky. It also contains the styling elements of the menu.
   - 'body' id selector contains the dimensions of the complete webpage. All elements in the webpage are inside the body.
   - 'content' id selector contains the style properties of the content section of all pages.
   - 'header' and 'footer' element selectors that have all styling elements for headers and footers.
   - 'section' element selector is used to divide the page into separate columns.

  Specific style sheets for each page

   - Style sheet for Home page: 'home page.css'.
   - Style sheet for Now showing page: 'Now showing theatre.css'.
   - Style sheet for Tickets page: 'Ticket page.css'.
   - Style sheet for Print page: 'Print page.css'.

  Note: No embedded styles or inline styles have been used.
